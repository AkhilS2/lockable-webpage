# Lockable Landing Page

Landing page for the Lockable iOS app.

## Setup

1. Clone the repository
2. Add your app screenshots to the `assets/` folder:
   - `screenshot-1.png`
   - `screenshot-2.png`
   - `screenshot-3.png`
   - `screenshot-4.png`

## Opening the Website

### Option 1: Open directly
Open `index.html` in your browser by double-clicking the file or dragging it into a browser window.

### Option 2: Local server
Run a local server from the project directory:

```bash
python3 -m http.server 8080
```

Then visit [http://localhost:8080](http://localhost:8080) in your browser.
